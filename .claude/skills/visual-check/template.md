# Шаблон отчёта visual-check

## Один вьюпорт

```
## Визуальная проверка: <url> (<viewport>)

### Статус: 🟢 OK / 🟡 Предупреждения / 🔴 Ошибки

### Скриншот
<встроен выше>

### Находки
- 🔴 ...
- 🟡 ...
- 🟢 Страница загружена, контент отображается корректно

### Консоль
<ошибки и предупреждения или "чисто">

### DOM
<заголовок страницы, основные секции, интерактивные элементы>
```

## Несколько вьюпортов или страниц — сводная таблица

```
## Итог: <url или "все страницы">

| Страница   | mobile | tablet | desktop |
|------------|--------|--------|---------|
| /          | 🟢     | 🟢     | 🟡      |
| /login     | 🟢     | 🟢     | 🟢      |

<критичные проблемы и паттерны по вьюпортам>
```

**Правила:**

- Скриншот всегда встраивается перед разделом «Находки».
- Если ошибок нет — раздел «Консоль» заменяется строкой «чисто».
- Сводная таблица — только если проверялось больше одного вьюпорта или страницы.
