# API-справочник

Базовый URL: `http://localhost:3001`  
Swagger UI: `http://localhost:3001/api/docs`

Все эндпоинты, кроме `POST /auth/register` и `POST /auth/login`, требуют заголовок:
```
Authorization: Bearer <accessToken>
```

---

## Auth `/auth`

### `POST /auth/register`

Регистрация нового пользователя.

**Тело запроса:**
```json
{
  "name": "Олег Скареднов",
  "email": "user@example.com",
  "password": "secret123"
}
```

**Ответы:**

| Статус | Описание |
|---|---|
| `201` | Пользователь создан |
| `400` | Невалидные данные (не email, пароль < 8 символов) |
| `409` | Email уже занят |

```json
{
  "accessToken": "eyJhbGci...",
  "user": { "id": "uuid", "name": "Олег Скареднов", "email": "user@example.com" }
}
```

---

### `POST /auth/login`

Вход по email и паролю. Возвращает `200` (не `201`).

**Тело запроса:**
```json
{ "email": "user@example.com", "password": "secret123" }
```

**Ответы:**

| Статус | Описание |
|---|---|
| `200` | Успешный вход |
| `401` | Неверный email или пароль |

Ответ аналогичен `/auth/register`.

## Categories `/categories`

Все маршруты защищены JWT.

### `POST /categories`

Создать категорию.

**Тело запроса:**
```json
{ "name": "Продукты", "color": "#4CAF50", "icon": "shopping-cart" }
```

Формат `color` — строго `#RRGGBB`.

**Ответы:**

| Статус | Описание |
|---|---|
| `201` | Категория создана |
| `400` | Невалидные данные |
| `401` | Не авторизован |
| `409` | Категория с таким именем уже существует |

```json
{
  "id": "uuid",
  "name": "Продукты",
  "color": "#4CAF50",
  "icon": "shopping-cart",
  "userId": "uuid",
  "createdAt": "2024-06-01T00:00:00.000Z",
  "updatedAt": "2024-06-01T00:00:00.000Z"
}
```

---

### `GET /categories`

Список всех категорий текущего пользователя.

**Ответы:**

| Статус | Описание |
|---|---|
| `200` | Массив категорий |
| `401` | Не авторизован |

Возвращает `Category[]`.

---

### `GET /categories/:id`

Категория по UUID.

**Параметры пути:** `id` — UUID категории.

**Ответы:**

| Статус | Описание |
|---|---|
| `200` | Категория найдена |
| `401` | Не авторизован |
| `404` | Категория не найдена |

---

### `PATCH /categories/:id`

Частичное обновление категории. Все поля тела опциональны.

**Тело запроса:**
```json
{ "name": "Кафе", "color": "#FF5722", "icon": "utensils" }
```

**Ответы:**

| Статус | Описание |
|---|---|
| `200` | Категория обновлена |
| `400` | Невалидные данные |
| `401` | Не авторизован |
| `404` | Категория не найдена |
| `409` | Новое имя уже занято |

---

### `DELETE /categories/:id`

Удалить категорию. Запрещено при наличии связанных транзакций.

**Ответы:**

| Статус | Описание |
|---|---|
| `204` | Удалено |
| `401` | Не авторизован |
| `404` | Категория не найдена |
| `409` | Есть связанные транзакции |

## Transactions `/transactions`

Все маршруты защищены JWT.

### `POST /transactions`

Создать транзакцию.

**Тело запроса:**
```json
{
  "amount": "1500.00",
  "type": "EXPENSE",
  "description": "Обед в кафе",
  "date": "2024-06-15",
  "categoryId": "uuid"
}
```

Поля: `amount` — строка Decimal 12,2; `type` — `INCOME` | `EXPENSE`; `description` — опционально, до 255 символов; `date` — ISO date `YYYY-MM-DD`.

**Ответы:**

| Статус | Описание |
|---|---|
| `201` | Транзакция создана |
| `400` | Невалидные данные или несуществующий `categoryId` |
| `401` | Не авторизован |

```json
{
  "id": "uuid",
  "amount": "1500.00",
  "type": "EXPENSE",
  "description": "Обед в кафе",
  "date": "2024-06-15T00:00:00.000Z",
  "categoryId": "uuid",
  "userId": "uuid",
  "createdAt": "2024-06-15T12:00:00.000Z",
  "updatedAt": "2024-06-15T12:00:00.000Z"
}
```

---

### `GET /transactions`

Постраничный список транзакций с фильтрами.

**Query-параметры:**

| Параметр | Тип | Описание |
|---|---|---|
| `dateFrom` | `YYYY-MM-DD` | Начало диапазона (включительно) |
| `dateTo` | `YYYY-MM-DD` | Конец диапазона (включительно) |
| `type` | `INCOME \| EXPENSE` | Фильтр по типу |
| `categoryIds[]` | `uuid[]` | Фильтр по категориям (можно повторить) |
| `page` | `number ≥ 1` | Номер страницы (дефолт: 1) |
| `limit` | `1–100` | Размер страницы (дефолт: 10) |

**Ответ `200`:**
```json
{
  "items": [ /* Transaction[] */ ],
  "total": 42,
  "page": 1,
  "limit": 10
}
```

---

### `GET /transactions/summary`

Агрегированная сводка. Принимает те же query-параметры фильтрации, что и список.

> Объявлен **до** `/:id` — иначе строка "summary" попадает в параметр id.

**Ответ `200`:**
```json
{
  "totalIncome": "50000.00",
  "totalExpense": "32500.00",
  "balance": "17500.00",
  "transactionCount": 15,
  "byCategory": [
    {
      "categoryId": "uuid",
      "totalIncome": "0.00",
      "totalExpense": "12000.00",
      "transactionCount": 5
    }
  ]
}
```

---

### `GET /transactions/monthly`

Помесячная статистика. Пустые месяцы в диапазоне `dateFrom`–`dateTo` заполняются нулями.

> Объявлен **до** `/:id`.

**Ответ `200`:**
```json
[
  { "month": "2024-04", "income": "50000.00", "expense": "32500.00" },
  { "month": "2024-05", "income": "50000.00", "expense": "28000.00" },
  { "month": "2024-06", "income": "0.00",     "expense": "12000.00" }
]
```

---

### `GET /transactions/:id`

Транзакция по UUID.

**Ответы:**

|---|---|
| `200` | Транзакция найдена |

---
